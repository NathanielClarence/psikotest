# Psikotest

Aplikasi psikotes untuk penerimaan pegawai baru. 
Untuk sementara aplikasi hanya dapat digunakan secara lokal. 
Integrasi dengan basis data dalam proses.

# Library used:
- PyQt5 5.14.1
- pyqt5-tools 5.14.0
- xlsxwriter 1.2.7
- Compiled using pyinstaller 3.6
